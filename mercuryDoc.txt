Mecury: Package Manager for Halo Custom Edition

-- DOCUMENTATION --

Commands available:

install     Look and search for packages hosted in Mercury repos to download and install in the game.
            TAKES:
            
            <packageName> <subPackageName> (optional)
            

remove      Delete previously installed packages in the game.

update      Update an existent package in the game.

mitosis     Create a new instance of Halo Custom Edition with only neccesary base files to run.

config      Define critical Mercury parameters as package installation path.

version     Throw version and related info about Mercury.
