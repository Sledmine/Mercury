Mercury 1.0


¿Qué es Mercury?

Es un administrador de paquetes para Halo Custom Edition.

¿Qué es un paquete?

Un paquete es algo que puede ser agregado al juego, una extensión un mapa, algún parche, mods, scripts, etc.

¿Qué necesito para usarlo?

Una versión de Halo Custom Edition 1.10 INSTALADA en nuestro PC (de preferencia nueva).

¿Cómo lo instalo?

1.- Descargar el siguiente archivo .zip

2.- Descomprimir el archivo en cualquier carpeta (de preferencia "Mis Documentos").

3.- Entrar a la carpeta llamada "Mercury1.0"

4.- Abrir una linea de comando con permisos de administrador en esa carpeta.

¿Cómo se usa?

Luego de abrir la linea de comando como administrador puedes usar los siguientes comandos:

mercury install

mercury remove

mercury list

mercury version

Para obtener información acerca de ellos escribe "mercury" en la consola.

¿Cómo descargo Flood 09?

Simplemente escribe el comando:

mercury install f09
