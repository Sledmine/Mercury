You can install any package by using this line on the Mercury console: 
mercury install <packageLabel>
Mercury is a simple package manager created with the Lua programming language by modders for modders, the main goal of it is to provide an easy way to release mods and provide updates between them, it is being developed and mantained under the GNU General Public License v3.
